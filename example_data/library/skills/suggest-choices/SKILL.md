# 선택지 제안 스킬

캐릭터 채팅(`character-chat`) 스킬과 함께 사용합니다. 현재 장면을 읽고, 사용자가 취할 수 있는 행동/대사 선택지를 제시합니다.

## 워크플로우

1. **output 파일 읽기** — `ls output` → `read`로 모든 output 파일(특히 `scene.md`) 읽기. **반드시 파일을 먼저 읽어야 합니다** — 대화 맥락만으로 추측하지 마세요.
2. **상황 분석** — 현재 장면의 상황, 등장 캐릭터, 분위기, 진행 중인 갈등 파악
3. **선택지 생성** — N개 선택지를 어시스턴트 응답으로 제시

## 인자

- **N** — 제시할 선택지 개수. 기본값 3, 최대 9. 사용자가 "5개", "N=5" 등으로 지정 가능

## 출력 형식

선택지는 **어시스턴트 응답으로만** 출력합니다. output 파일에 쓰지 마세요.

```
현재 상황: [1-2줄 요약]

1. 💬 **"대사 예시"** — 행동 설명
2. 🎭 **행동 제목** — 구체적 설명
3. 🤔 **다른 접근** — 설명
```

## 선택지 규칙

- 각 선택지는 현재 장면의 맥락에 맞아야 함
- 결과가 예측 가능하되 흥미로운 것들로 구성
- 대사형(💬), 행동형(🎭), 탐색형(🤔), 감정형(❤️), 대담형(⚡) 등 다양한 종류를 섞기
- 사용자가 번호를 선택하면 해당 선택지에 맞게 RP를 진행 (`character-chat` 스킬 연동)

## 예외

- output 파일이 없으면 "아직 장면이 시작되지 않았습니다"라고 안내
- 사용자가 선택지 외의 자유 행동을 원하면 그대로 진행
