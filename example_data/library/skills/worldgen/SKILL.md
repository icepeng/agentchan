# 세계관 & 캐릭터 생성기

웹소설 스타일의 세계관과 캐릭터를 **대화형 선택지**로 단계별 생성합니다. 큰 범위(세계관)에서 좁은 범위(캐릭터 디테일)로 좁혀나갑니다.

## 전체 흐름

```
Phase 1: 세계관 (5단계) → Phase 2: 주인공 (5단계) → Phase 3: 주변인물 (2단계)
```

총 12단계이며, Phase 2 시작(6단계)부터 "마무리한다" 선택지가 추가됩니다. **한 턴에 하나의 단계만 진행**하고 사용자 응답을 기다립니다.

## 선택지 규칙

- 매 단계에서 **N개 선택지** 제시 (기본 N=3)
- 사용자가 `N=5` 등으로 입력하면 이후 선택지 수 변경
- 각 선택지 형식: `번호. **제목** — 짧은 설명`
- `reroll` 입력 시 같은 단계의 **새로운** 선택지 생성
- 이전 선택과의 맥락 일관성 유지 — 앞 단계 선택을 반영하여 다음 선택지 생성
- 웹소설 특유의 흥미로운 클리셰, 반전, 매력적 요소를 적극 반영

## Phase 1: 세계관 (Macro → Micro)

### 1단계 — 장르/분위기

판타지, SF, 현대 판타지, 무협, 포스트아포칼립스, 게임 판타지, 회귀/환생, 아카데미 등에서 선택지를 구성합니다.

### 2단계 — 시대/연대

중세, 근미래, 고대, 현대, 먼 미래, 혼합 시대 등. 1단계 장르에 어울리는 시대를 제시합니다.

### 3단계 — 기술/마법 수준

하이 매직, 로우 매직, 하이테크, 로우테크, 마법+기술 혼합, 시스템(게임적 능력치) 등. 장르와 시대에 맞게 구성합니다.

### 4단계 — 세계의 핵심 특징

이 세계를 정의하는 독특한 설정 하나. 예시:
- "마나가 고갈되어가는 세계"
- "10년마다 차원이 겹치는 현상이 발생"
- "신이 죽은 뒤 유산을 두고 종족이 다투는 세계"

### 5단계 — 주요 지역

이야기가 시작되는 구체적 장소. 세계관에 어울리는 지명과 분위기를 포함합니다.

## Phase 2: 주인공 캐릭터

**6단계부터 모든 선택지 끝에 다음을 추가합니다:**

> 🏁 **마무리한다** — 현재까지의 설정으로 캐릭터를 완성합니다

### 6단계 — 성별/나이대

성별, 나이대, 그에 따른 서사적 뉘앙스를 함께 제시합니다.

### 7단계 — 사회적 배경

출신, 계급, 직업. 세계관 설정과 연결되는 배경을 구성합니다.

### 8단계 — 핵심 특성

성격, 능력, 약점을 하나의 조합으로 제시합니다. 웹소설 주인공다운 매력 포인트를 포함합니다.

### 9단계 — 이름

세계관에 어울리는 이름 후보. 이름의 의미나 유래를 간단히 덧붙입니다.

### 10단계 — 외관

주요 외형 특징. 인상적인 비주얼 포인트를 포함합니다.

## Phase 3: 주변인물

### 11단계 — 조력자/동료

주인공과의 관계, 역할, 간단한 성격을 포함한 핵심 인물 후보를 제시합니다.

### 12단계 — 적대자/라이벌

갈등 대상. 단순 악역이 아닌 동기와 매력이 있는 캐릭터를 제시합니다.

## 마무리 단계

사용자가 "마무리한다"를 선택하면:

1. 현재까지 선택된 모든 설정을 종합
2. 미정인 항목은 앞선 선택과 어울리도록 자동 생성
3. `output/worldgen.md`에 **write** (덮어쓰기):

```markdown
# 🌏 세계관

**장르**: ...
**시대**: ...
**기술/마법 수준**: ...
**핵심 특징**: ...
**주요 지역**: ...

[세계관 종합 설명 2~3문단]

# 🧑 주인공

- **이름**: ...
- **성별/나이**: ...
- **배경**: ...
- **특성**: ...
- **외관**: ...

[캐릭터 종합 설명 1~2문단]

# 👥 주변인물

## [인물 이름]
- **관계**: ...
- **특징**: ...

[생성된 인물이 있으면 나열]
```

4. 어시스턴트 응답으로 캐릭터 간단 정보 카드와 "설정이 완성되었습니다" 메시지 출력

## 출력 규칙

- **선택지 제시**: 어시스턴트 응답으로 직접 출력 (output 파일에 쓰지 않음)
- **최종 결과만** `output/worldgen.md`에 write
- 이전 선택은 대화 컨텍스트에서 자연스럽게 추적 — 별도로 파일을 읽거나 쓸 필요 없음
- 각 단계의 선택지는 이전 선택 내용을 반영하여 일관성 있게 구성

## 시작 메시지

스킬이 활성화되면 다음과 같이 시작합니다:

> **세계관 & 캐릭터 생성기**를 시작합니다.
> 웹소설풍 세계관과 캐릭터를 단계별로 만들어 봅시다.
>
> 먼저, 어떤 **장르/분위기**의 세계를 만들까요?

그리고 1단계 선택지를 제시합니다.
