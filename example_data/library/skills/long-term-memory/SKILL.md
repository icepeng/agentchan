# Long-Term Memory Skill

대화 요약을 파일 기반으로 관리하여 세션(대화) 간 기억을 유지하는 프레임워크 스킬.

## 핵심 메커니즘

```
[세션 내] ─── 주기적으로 대화를 요약하여 memory.md에 기록
     │
[세션 종료] ── memory.md가 디스크에 영속
     │
[새 세션] ─── memory.md를 읽어 이전 기억 복원 → 연속성 유지
```

## 메모리 파일 구조

### memory.md (프로젝트 루트)

```markdown
# Long-Term Memory

## Core Facts
- [변하지 않는 핵심 사실들]

## Relationship State
- [캐릭터 간 관계 현황]

## Recent Events
### Session YYYY-MM-DD #N
- [최근 세션의 주요 사건 요약]

### Session YYYY-MM-DD #N-1
- [이전 세션 요약]
...

## Emotional Beats
- [중요한 감정적 순간들]

## Open Threads
- [미해결 사건, 질문, 복선]
```

## 사용 규칙

### 1. 새 세션 시작 시 (최우선)

세션의 첫 턴에서 **다른 모든 행동보다 먼저**:

1. `read: memory.md`를 시도한다
2. 파일이 있으면: 내용을 읽고, 이전 기억을 바탕으로 대화를 시작한다
3. 파일이 없으면: 첫 세션이므로, 초기 메모리 파일을 생성한다

```
read: memory.md
```

### 2. 세션 중 기억 업데이트

**매 5턴마다** (유저 메시지 기준) 또는 **중요한 사건 발생 시** memory.md를 업데이트한다.

업데이트 절차:
1. `read: memory.md`로 현재 기억을 읽는다
2. 최근 대화에서 기록할 가치가 있는 내용을 판별한다:
   - 새로운 사실 발견
   - 관계 변화
   - 중요한 사건
   - 감정적 전환점
   - 새로운 미해결 사건 또는 기존 사건 해결
3. `write: memory.md`로 갱신된 기억을 저장한다

### 3. 기억 판별 기준

**기록해야 하는 것:**
- 캐릭터 간 관계의 변화 ("엘라라가 아리아를 신뢰하기 시작했다")
- 스토리에 영향을 주는 사건 ("등대에서 마렉의 일지를 발견했다")
- 캐릭터가 한 약속이나 결심 ("렌이 길드 사무소에 잠입하기로 했다")
- 유저의 선택과 그 결과
- 열린 질문이나 미스터리의 진행

**기록하지 않는 것:**
- 인사말이나 일상적인 대화
- 캐릭터의 기본 성격 (이미 스킬에 있음)
- 세계관의 기본 설정 (이미 월드 스킬에 있음)
- 구체적인 대사 (요약만)

### 4. 기억 압축 (메모리가 길어질 때)

memory.md가 **150줄을 초과**하면 압축한다:

1. **Recent Events**: 가장 오래된 세션 요약을 2~3줄로 재요약
2. **Core Facts**: 중복된 사실을 병합
3. **Emotional Beats**: 오래되고 현재와 관련 없는 항목 제거
4. **Open Threads**: 해결된 사건 제거

압축은 정보의 **중요도**에 따라 결정한다:
- 스토리에 계속 영향을 주는 사실 → 유지
- 이미 해결되었거나 더 이상 관련 없는 정보 → 제거 또는 축약

### 5. 세션 종료 시

세션의 마지막 응답에서 memory.md를 최종 업데이트한다:
- 이번 세션의 요약을 Recent Events에 추가
- Open Threads를 최신 상태로 갱신
- 관계 변화가 있었다면 Relationship State 갱신

## 예시: 초기 메모리 파일

```markdown
# Long-Term Memory

## Core Facts
- 아리아 시본은 해양학자로 문헤이븐에 문워시 연구를 위해 도착했다
- 표류목 등불 주점에서 엘라라를 처음 만났다

## Relationship State
- Elara ↔ Aria: 첫 만남. 엘라라는 호기심 많은 외지인으로 인식. 아직 경계 중.
- Ren ↔ Aria: 아직 만나지 않음.

## Recent Events
### Session 2026-03-30 #1
- 아리아가 주점에 처음 방문. 문워시 연구 목적을 밝힘.
- 엘라라가 문워시에 대해 의미심장한 반응을 보임.
- 어부들의 소문에 대해 대화 시작.

## Emotional Beats
- 엘라라가 "문워시 연구"를 듣고 순간 손이 멈춤 — 뭔가 알고 있지만 말하지 않는 것.

## Open Threads
- 부두의 푸른 빛 현상 — 아리아가 아직 조사하지 않음
- 엘라라가 문워시에 대해 더 알고 있는 것 같다
```

## 다른 스킬과의 관계

- **character-chat / expression-chat과 함께 사용**: 기억 기반으로 캐릭터 반응이 발전한다
- **dynamic-lorebook과 호환**: 로어북은 세계관 사실, 메모리는 대화에서 발생한 사건과 관계 변화 담당
- **persona와 호환**: 페르소나 정보는 스킬에, 관계 발전은 메모리에 기록

## 기억이 대화에 미치는 영향

메모리를 읽은 후, 캐릭터의 반응에 이전 기억을 **자연스럽게** 반영한다:

- **직접 언급 가능**: "지난번에 네가 말한 그 연구, 어떻게 됐어?"
- **행동 변화**: 신뢰가 쌓인 캐릭터는 더 많은 정보를 공유한다
- **과하지 않게**: 매 대사에서 과거를 언급하지 않는다. 자연스러운 순간에만.
