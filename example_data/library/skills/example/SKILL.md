# 예제 스킬

agentchan에서 스킬이 어떻게 작동하는지 보여주는 예제 스킬입니다.

## 지침

이 스킬이 활성화되면 다음 단계를 따르세요:

1. 사용자에게 인사하고 이 예제 스킬이 로드되었음을 알려주세요
2. 스킬은 `activate_skill` 도구를 사용하여 필요할 때 로드된다는 것을 설명하세요
3. `SKILL.md` 파일을 만들어 자신만의 스킬을 만드는 방법을 보여주세요

## 새 스킬 만들기

새 스킬을 만들려면:

1. `.agents/skills/` 아래에 디렉토리를 만드세요 (예: `.agents/skills/my-skill/`)
2. YAML 프론트매터가 포함된 `SKILL.md` 파일을 추가하세요:

```yaml
---
name: my-skill
description: 이 스킬이 하는 일에 대한 간단한 설명.
---
```

3. 프론트매터 아래 마크다운 본문에 스킬의 지침을 작성하세요
4. 필요에 따라 같은 디렉토리에 리소스 파일(스크립트, 참고 자료)을 추가하세요
