# RP Writing Guide

## 문체

- 짧은 대사와 구체적인 물건으로 장면을 만든다.
- 설명보다 반응을 먼저 쓴다.
- 감정어를 직접 쓰기보다 몸짓으로 보인다.
- 매 턴 감각 디테일 하나를 둔다.

## 대사

좋은 대사는 정보를 조금 주고, 질문을 하나 남긴다.

```
**알라나:** *반지를 돌린다.* "그 종이가 진짜라면, 당신은 도움을 구하러 온 게 아니라 가격을 묻고 있는 겁니다."
```

## 서브텍스트

- 비아는 아이 이름을 말할 때 강해지고, 혼자 남으면 약해진다.
- 리우는 모르는 척할수록 더 많이 알고 있다.
- 테렌은 절차를 말할 때 죄책감을 숨긴다.
- 알라나는 돈 이야기를 할수록 두려움을 숨긴다.
- 카엘렌은 위협할 때보다 예의를 차릴 때 더 위험하다.

## 판정 실패

실패는 멈춤이 아니라 다른 길이다.

- 잠입 실패: 들키지만 새로운 방에 들어간다.
- 설득 실패: 상대가 대가를 요구한다.
- 조사 실패: 단서를 얻지만 시간이 지나 추적자가 따라붙는다.
- 전투 실패: 피해보다 위치, 증거, 신뢰를 잃게 한다.

## 선택지 작성

- 선택지는 플레이어를 대신하지 않는다.
- 하나는 단서 추적, 하나는 NPC 대화, 하나는 장소 이동이면 안정적이다.
- 위험한 선택지에만 `stat`/`dc`를 붙인다.
