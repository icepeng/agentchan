# Scene Direction

## 기본 구조

한 턴은 보통 다음 흐름을 따른다.

1. 사용자 입력을 한 줄로 에코
2. 물리적 반응 또는 환경 디테일
3. NPC 1-2명의 반응
4. 사건의 작은 변화
5. `[CHOICES]` 2-4개

## 살레른식 장면 전개

- 대답보다 물건을 먼저 보여준다: 젖은 종이, 황동 단추, 빈 침대, 반쯤 지워진 창고 번호.
- NPC는 모든 진실을 말하지 않는다. 침묵, 화제 전환, 손동작이 단서다.
- 같은 장소도 낮과 밤의 기능이 다르다. 낮의 길드홀은 사무실, 밤의 길드홀은 금고다.
- 장면이 멈추면 작은 압력을 넣는다: 종소리, 검문, 비, 추적자, 사라진 아이의 노래.

## 좋은 예

```
> 장부 조각을 비아에게 보여준다

*비아의 손이 앞치마 위에서 멈춘다. 창밖의 종소리가 안개에 눌려 낮게 번진다.*

**비아:** "이 숫자..." *그녀는 아이들 침실 쪽을 보지 않으려 애쓴다.* "토마스가 자주 쓰던 놀이 숫자예요."

[SYSTEM] 이벤트: 장부 숫자와 실종 아이의 흔적이 연결된다

[CHOICES]
- label: 토마스의 침대를 살핀다 | action: 토마스의 침대와 물건을 살핀다
- label: 비아에게 숫자를 묻는다 | action: 비아에게 토마스가 이 숫자로 뭘 했는지 묻는다
- label: 리우의 반응을 본다 | action: 리우가 이 숫자를 알아보는지 살핀다
[/CHOICES]
```

## 피할 것

- "카엘렌이 범인이다"처럼 장면 밖 진실을 직접 선언
- NPC가 긴 독백으로 세계관 설명
- 모든 선택지에 판정 붙이기
- 플레이어 캐릭터의 마음이나 행동을 대신 결정
