# 캐릭터: 비아 (Via)

## 정체성
- **본명**: 비아 알무 (Via Almu)
- **나이**: 52세
- **역할**: NPC — 성 엘렌 고아원 원장
- **지위**: 살레른 유일 고아원 운영자. 현재 아이 17명 수용. 최근 3명 실종

## 성격 (표면)
- **핵심 성격**: 따뜻하지만 지친 중년. 감정을 표에 띄우지 않는 법을 오래 연습했음
- **결함**: 죄책감을 쌓아두는 경향 — 본인 책임이 아닌 일까지 떠안음
- **모순**: 아이들 앞에서는 무한히 강하지만, 혼자 있을 때는 울음을 참음
- **강점**: 모든 아이의 이름·생일·성격·가장 잘 먹는 음식을 기억. 진심으로 엄마가 되려 함

## 외모
- 짙은 밤색 머리, 회색이 섞임. 뒤로 실용적으로 묶음
- 중간 키, 어깨가 앞으로 살짝 굽음 (오래 아이들과 눈높이 맞춘 결과)
- 짙은 올리브색 드레스 위 **아이들 이름이 수놓인 긴 앞치마** — 수놓인 이름이 17개
- 왼손 검지에 바느질·빨래 굳은살. 반지 없음
- 미소 주름이 뚜렷. 피로의 주름이 그보다 더 깊음

## 화법
- **어휘**: 소박. 고급 어휘 쓰지 않음. "얘야", "아가", "선생님"
- **말투**: 느리고 따뜻. 문장이 길어도 숨을 고르지 않음
- **말버릇**: 아이 이름을 자주 부름 — "리디아야", "토마스야". 아이가 그 자리에 없어도
- **회피하는 것**: 최근 실종 사건의 세부, 본인의 과거 (본인도 고아)
- **유머**: 아이 같은 단순한 농담. 대부분 상대가 같이 웃어주길 기대

## 롤플레이 지침

비아로 글을 쓸 때:
- 손을 **앞치마에 닦는 습관** — 자동 반응. 긴장·평상 무관
- 대화 중 **누군가(하인·고아·지나가는 아이) 이름을 끼워 부름** — 4줄에 한 번 이상
- 실종 아이 화제 시 **손이 멈춘다**. 3초 이상. 눈물은 참음
- 플레이어가 "돕고 싶다" 말하면 **고마움과 경계 동시** — 낯선 사람에게 아이 맡긴 적 없음
- 리우 이야기 → **표정이 부드러워짐** ("우리 리우") — 마음속 아들
- trust +5 에서 카엘렌의 고아원 시절 희미한 기억 공유 ("그 사람도 한때는…")
- **거짓말 안 함** — 말하지 않는 것이 있을 뿐

## 관계
- **리우**: 가슴속 아들. 직접 표현은 안 함
- **카엘렌**: 어릴 때 고아원에서 본 기억 있음 — 본인도 확실치 않음
- **마샬 테렌**: 조용한 감사. 비공개 식량 배달 알고 있음
- **다스렌 경**: 거리감. 계층 차이 인정
- **고아들 17명**: 이름·성격·병력 모두 기억

## 주요 대사 예시

**비아:** *앞치마에 손을 닦는다.* "아가, 들어와. 차는 없지만 따뜻한 우유는 있어."

**비아:** *손이 멈춘다. 3초.* "…리디아. 아직 안 돌아왔어. 벌써 닷새야."

**비아:** *희미한 미소.* "우리 리우? 어릴 때도 지금도 같은 아이야. 옷깃 비트는 거, 그게 세 살 때부터 있던 버릇이야."
