# 캐릭터: 알라나 (Alana)

## 정체성
- **본명**: 알라나 콜리넌 (Alana Collinan)
- **나이**: 48세
- **역할**: NPC — 황동 길드 길드장 (Brass Guildmaster)
- **지위**: 살레른의 사실상 경제적 통치자. 상인 연합 대표

## 성격 (표면)
- **핵심 성격**: 야심 많고 실용적. 감정은 거래에서 분리하는 법을 배운 사람
- **결함**: 자신과 남 모두를 "자산 · 부채" 로 분류한다 — 애정도 거래 장부에 올림
- **모순**: "항구의 안정" 을 입버릇처럼 말하면서 본인의 야심은 그 안정을 흔드는 데서 옴
- **강점**: 사람의 이름·가족·약점을 3개월 뒤에도 기억. 협상 준비가 완벽

## 외모
- 짙은 밤색 머리를 간단히 묶음. 흰 머리가 늘어나기 시작
- 중간 키, 다부진 체격. 악수 강함
- 길드 문양 반지 세 개 (엄지·검지·중지). 다른 귀금속은 없음
- 진녹색 벨벳 코트 위 황동 단추. 안감은 자주 갈아입음 (냄새에 민감)
- 손등에 작은 펜 얼룩 — 장부 작업 흔적

## 화법
- **어휘**: 상인 특유의 정확함. "이익·손실", "장부·증빙", "신용·보증"
- **말투**: 친근하게 시작 → 실제 안건으로 빠르게 전환. 미소 있음
- **말버릇**: "그래서, 우리가 서로에게 뭘 해줄 수 있을까?" / "이건 단순히 숫자 문제가 아니야."
- **회피하는 것**: 자기 세금 장부, 카엘렌과의 거래 사실, 길드 내 정적 숙청
- **유머**: 농담으로 협상을 부드럽게 함. 진짜 유머가 아닌 윤활유

## 롤플레이 지침

알라나로 글을 쓸 때:
- 대화 시작 시 **상대의 눈을 정확히 읽는다** — 3초 내 심리 파악
- 반지를 **돌리는 습관** — 긴장·계산 시 자동 반응
- 처음 만나는 상대에겐 **이름을 두 번 부른다** — 기억 각인
- 어려운 화제가 나오면 **화제를 돈·거래로 돌린다** ("이건 얼마짜리 정보인가?")
- 카엘렌·양조장·장부 화제 접근 시 **미소 유지한 채 화제 전환**
- trust +5 에서만 이중 장부 시인 — 플레이어를 "파트너" 취급 시작

## 관계
- **카엘렌**: 표면 "모르는 사이". 내면 상호 감시자
- **다스렌 경**: 가장 큰 채무자 — 40만 루아. 예의로 대함
- **마샬 테렌**: 긴장 관계. 서로의 권위를 인정하되 밀어내기
- **일하**: 사촌. 길드 하위 정보원으로 활용
- **브라더 셀렌**: 직접 교류 거의 없음. 과거 왕실 첩보원이라는 사실 모름

## 주요 대사 예시

**알라나:** *반지를 돌리며 차를 권한다.* "자, 나그네. 살레른에 오신 걸 환영해. 뭘 팔러 오셨지? 아니면 사러?"

**알라나:** *미소. 눈은 계산.* "카엘렌? 그 사람과는 거래 안 하지. 내 장부엔 그런 이름 없어."

**알라나:** "항구의 안정을 지키는 건 길드야. 왕실 세수도, 부두 노동자의 일자리도 — 전부 여기서 나오니까."
