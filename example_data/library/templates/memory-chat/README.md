# 메모리 챗

**Draft: 아직 완성되지 않았습니다. 구성 참고용으로만 사용하는 것을 권장합니다.**

세션이 끝나도 주요 사건을 `files/memory/journal.md` 에 기록해두고, 다음 세션에서 그 기억을 참조합니다. 여러 번에 걸쳐 이어지는 장편 RP에 적합합니다.

## 이렇게 쓰세요

- **저널** — `files/memory/journal.md` 는 append-only 입니다. "언제 무슨 일이 있었다"를 한 줄씩 쌓습니다.
- **캐릭터/페르소나/세계관/장면** — 캐릭터 챗과 동일합니다.
- **감정 삽화** — 감정 전환 시점에 인라인 삽화(이미지)가 들어갈 수 있도록 연출됩니다.

오래 이어진 RP에서는 저널 덕분에 "지난 회에 누구랑 싸웠는지", "지난 주에 무슨 약속을 했는지" 같은 맥락이 살아납니다.

## 시작하는 한 줄

- **지난 저널을 읽고, 거기에 이어서 오늘을 시작**
- **엘라라와의 관계가 애매한 상태에서 다시 만나는 장면**
- **3일 뒤. 지난 사건의 후유증이 남아 있는 아침**

## 더 알아보기

- 저널이 너무 길어지면 `/compact` 명령으로 요약해 새 세션으로 이어갈 수 있습니다.
- 기억을 리셋하고 싶으면 Edit Mode에서 `files/memory/journal.md` 를 비우면 됩니다.
