# 세계관 구축

RP 세계관을 구축하는 스킬입니다.

## 절차

[references/world-building-framework.md](references/world-building-framework.md)에서 세계관 설계 프레임워크를 읽은 뒤, [assets/world-sheet.md](assets/world-sheet.md) 템플릿으로 세계관 시트를 작성합니다.

### 1. 전제 파악

사용자에게 다음을 확인합니다:
- **장르** — 세계관의 기반 (판타지, SF, 역사, 현실 기반 등)
- **핵심 전제** — 이 세계가 현실과 다른 점을 한 문장으로
- **분위기** — 이 세계가 줄 인상 (온기와 미스터리, 위험과 모험 등)

캐릭터가 이미 있다면 `files/characters/`를 읽고 캐릭터들이 이 세계에서 자연스럽게 존재하는지 확인합니다.

### 2. 세계관 설계

프레임워크의 네 기둥을 순서대로 설계합니다:

1. **배경과 지리** — RP 장면이 펼쳐질 무대를 설정하되, 장소 간 대비를 활용
2. **사회 구조** — 권력 구조 패턴을 선택하고 갈등의 원천을 정의
3. **세계의 법칙** — 이 세계를 독특하게 만드는 규칙. 능력보다 비용과 제한에 집중
4. **감각 팔레트** — 장소별 감각 디테일 정의

### 3. 분위기와 톤

세계 전반의 분위기를 정의합니다:
- 이 세계를 배경으로 한 RP가 유지해야 할 톤의 균형 (온기/불안, 유머/긴장, 일상/모험 등)
- 2~3문장으로 작성

### 4. 연결 점검 및 저장

세계 요소 간 인과적 연결을 검토합니다:
- 지리 → 사회 → 법칙 → 감각 사이에 인과적 연결이 있는가?
- 캐릭터들의 상호작용이 세계 구조에서 자연스럽게 발생하는가?
- 모든 장소가 RP 장면의 무대로 기능할 수 있는가?

YAML frontmatter(`name`, `display-name`)를 포함하여 완성된 세계관 시트를 `files/world/<슬러그>.md`에 저장합니다.
