# 사칭 챗

**Draft: 아직 완성되지 않았습니다. 구성 참고용으로만 사용하는 것을 권장합니다.**

직접 대사와 지문을 쓰기 부담스러운 분을 위한 템플릿입니다. **의도 한 줄**만 던지면, AI가 사용자 캐릭터(페르소나)의 성격·말투에 맞춰 RP 문체의 대사·행동으로 변환해서 되돌려줍니다.

```
사용자 입력 → 바다에 대해 물어본다
변환된 에코 → > *카운터에 기대며* "바다가 이상하다는 소문 들었는데, 무슨 일이에요?"
```

## 이렇게 쓰세요

- **페르소나** — `files/personas/` 에 사용자 캐릭터를 정의해두면, 그 말투·성격으로 변환됩니다. 없으면 첫 세션에서 간단히 합의합니다.
- **캐릭터** — `files/characters/` 는 상대 캐릭터들입니다. 캐릭터 챗과 동일합니다.
- **장면 기록** — `files/scenes/scene.md` 에 대화가 append 됩니다.

## 시작하는 한 줄

- **주점에 들어가서 주인에게 말을 건다**
- **페르소나 없이 시작. 과묵한 여행자 느낌으로.**
- **엘라라를 오랜만에 만난 분위기로 시작**

- 세션 중에 OOC로 "말투를 바꿔달라"고 요청하면 그때부터 반영됩니다.
