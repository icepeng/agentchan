# Last Vow

신부 **나유라**가 서약 직전 쓰러졌다. 공식 사인은 아직 나오지 않았지만, 모두가 독을 떠올린다.

문제는 당신이 **신부대기실에서 유라를 마지막으로 본 사람**이라는 점이다. 하객들은 웅성거리고, 웨딩홀 직원은 출입문을 막았고, 경찰은 90분 뒤 도착한다. 당신이 이 방을 나가기 전에 진짜 범인을 찾지 못하면, 이 결혼식의 범인은 당신으로 굳어진다.

세 명의 용의자가 있다.

- **서이서** — 웨딩 플래너. 모든 동선을 쥐고 있지만, 너무 침착하다.
- **박한결** — 신랑. 신부를 사랑한다고 말하지만, 방금 파혼 서류가 발견됐다.
- **윤민지** — 신부의 절친이자 축가 진행자. 가장 많이 울지만, 가장 많이 알고 있다.

## 시작하는 한 줄

- **문이 잠긴 폐백실에서 시작해줘**
- **내가 왜 의심받는지부터 보여줘**
- **세 명을 한 자리에 모으고, 유라가 마지막으로 마신 잔을 확인한다**

매 장면 끝에는 다음 행동 선택지가 표시됩니다. 버튼을 눌러 진행해도 되고, 자유롭게 질문하거나 추궁해도 됩니다.
