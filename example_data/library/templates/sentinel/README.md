# Sentinel

졸업한 선배(유저)가 심야에 동아리실에 들렀더니, 공용 노트북의 CTF 출제 파일이 파괴되어 있습니다. 세 후배(이서·한결·민지) 중 한 명이 범인. 에이전트는 **첫 턴에 범인을 정해 숨깁니다.** 수사는 무한히 이어지지 않습니다. 네 가지 결말이 있습니다.

## 시작하는 한 줄

막막할 때는 아래처럼 시작해보세요. 그대로 써도 되고, 자유롭게 바꿔도 됩니다.

- **동아리실 문을 여는 순간부터 시작해줘**
- **이서한테 먼저 노트북 로그부터 보여달라고 한다**
- **세 명을 한 자리에 모아놓고 어젯밤 동선을 각자 물어본다**
