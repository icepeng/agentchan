# 소설

장편 소설을 기획부터 교정까지 한 프로젝트에서 진행하는 템플릿입니다. 다섯 단계를 스킬로 분리해두었고, 스킬은 필요할 때만 `/skill-name` 으로 불러 씁니다.

## 이렇게 쓰세요

- **1. 전제 + 아웃라인** — `/outline` 으로 컨셉·장르·주제를 정의하고 플롯을 구성합니다.
- **2. 캐릭터** — `/characters` 로 주요 인물을 설계합니다.
- **3. 세계관** — `/world-building` 으로 배경 세계를 구축합니다.
- **4. 집필** — `/chapter-writing` 으로 아웃라인에 따라 챕터를 써 나갑니다.
- **5. 교정** — `/revision` 으로 일관성 점검, 문체 교정, 원고 편집을 진행합니다.

1~3단계는 순서를 바꿔도 되지만, 집필(4) 전에 아웃라인·캐릭터·세계관이 어느 정도는 갖춰져야 합니다.

## 시작하는 한 줄

- **판타지 로맨스의 아웃라인을 잡아보자. 주인공은 두 명.**
- **지난 세션에서 진행 상태부터 정리해줘**
- **`files/outline.md` 를 보고, 다음에 써야 할 챕터를 제안해줘**

## 더 알아보기

- 완성 원고는 `/revision` 의 compile 스크립트로 `files/manuscript.md` 에 모을 수 있습니다.
- 소설 제목으로 폴더를 따로 만들지 말고 `files/` 직하에 두세요 — 스킬들이 그 전제로 동작합니다.
