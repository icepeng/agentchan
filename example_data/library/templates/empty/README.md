# 빈 프로젝트

규격도, 약속된 파일 구조도 없는 백지 상태입니다. 뭘 하고 싶은지부터 에이전트와 같이 잡아가세요.

## 이렇게 쓰세요

- **SYSTEM.md** — 이 프로젝트가 "무엇인지" 적어두세요. 에이전트의 모든 응답이 이걸 기반으로 움직입니다.
- **files/** — 나머지 모든 콘텐츠가 여기에 들어갑니다. 폴더 구조는 자유입니다.
- **renderer.tsx** — `files/` 를 화면에 어떻게 보여줄지 결정하는 React 컴포넌트입니다. 기본 렌더러는 파일 목록만 찍어주니, 원하면 직접 고치세요.

## 시작하는 한 줄

- **이 프로젝트로 뭘 할 수 있을지 아이디어 3개만 던져줘**
- **캐릭터 한 명 만들고, 그 캐릭터랑 대화할 수 있게 설정해줘**
- **내 짧은 아이디어를 받아적을 아카이브로 쓰고 싶어**

## 더 알아보기

- 나중에 방향이 정해지면, 적합한 다른 템플릿(Character Chat, Novel 등)을 골라 새 프로젝트를 만드는 게 편할 수 있습니다.
