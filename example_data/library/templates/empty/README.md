# 빈 프로젝트

정해진 세계관이나 파일 구조가 없는 기본 상태입니다. 목적과 자료를
원하는 방식으로 채워 넣으세요.

## 구성

- **SYSTEM.md**: 프로젝트가 무엇인지, 에이전트가 어떻게 응답해야 하는지 적는 지침입니다.
- **files/**: 창작 자료와 출력물을 자유롭게 넣는 작업 공간입니다.
- **renderer/**: `files/`와 대화 상태를 화면으로 보여주는 Renderer V1 코드입니다.

## 시작 예시

- 이 프로젝트로 무엇을 만들 수 있을지 아이디어를 요청한다.
- 캐릭터 파일을 만들고 그 캐릭터와 대화할 수 있게 설정한다.
- 글 조각을 받아 아카이브 형태로 정리한다.
