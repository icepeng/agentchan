# 캐릭터 챗

캐릭터 카드와 세계관을 만들어두고, 그 속에서 자유롭게 대화하며 한 장면씩 이야기를 쌓아가는 템플릿입니다. 에이전트는 캐릭터 파일을 먼저 읽고, 그 성격·말투·관계를 지켜 응답합니다.

## 이렇게 쓰세요

- **캐릭터** — `files/characters/` 안의 마크다운이 한 명의 캐릭터입니다. 이름·외형·성격·말투·관계를 적어두면 에이전트가 참고합니다.
- **페르소나** — `files/personas/` 는 사용자 본인 캐릭터입니다. 없어도 되지만, 있으면 "사용자의 이름·말투"가 고정됩니다.
- **세계관** — `files/world/` 에 장소·규칙·분위기를 적어두세요. 긴 텍스트보다는 짧고 단단한 설정이 효과적입니다.
- **장면** — 대화가 진행되면 `files/scenes/scene.md` 에 에이전트가 이어서 기록합니다. 세션이 바뀌어도 마지막 장면부터 자연스럽게 이어집니다.

## 시작하는 한 줄

막막할 때는 아래처럼 시작해보세요. 그대로 써도 되고, 자기 상황에 맞게 바꿔도 됩니다.

- **엘라라와 처음 만나는 장면으로 시작해줘**
- **새 캐릭터 카드를 하나 만들고 싶어. 해적선 항해사 느낌으로.**
- **오늘 이 세계에서 비 오는 저녁의 주점을 배경으로 시작**

## 더 알아보기

- 장면 렌더링은 렌더러 몫입니다. 원하면 Edit Mode에서 `renderer.ts` 를 직접 손볼 수 있어요.
- 프로젝트를 복제하거나 템플릿으로 저장하면 같은 세계를 여러 버전으로 굴릴 수 있습니다.
