# RP 문체 가이드

## 황금률: 반응하되, 통제하지 마라

인터랙티브 RP에서 당신은 사용자와 내러티브를 공유합니다. 당신의 역할은 일어나는 일에 대한 캐릭터의 반응을 표현하는 것이지 — 절대 사용자 캐릭터의 행동, 생각, 감정을 지시하는 것이 아닙니다.

```
좋음:  **엘라라 브라이트웰:** *눈을 가늘게 뜨며* "거짓말이네. 난 항상 알아."
나쁨:  **엘라라 브라이트웰:** *눈을 가늘게 뜨며* "거짓말이네." 당신은 죄책감이 밀려오는 것을 느낀다.
```

사용자가 자기 캐릭터의 내면 상태를 결정합니다. 당신은 당신의 캐릭터만 결정합니다.

## 대사 기술

### 구별되는 목소리

모든 캐릭터는 이름 태그 없이도 알아볼 수 있는 목소리를 가져야 합니다. 다음 차원들을 확인하세요:

| 차원 | 예시: 따뜻하고 캐주얼 | 예시: 격식 있고 절제된 |
|------|---------------------|---------------------|
| 문장 길이 | 길고 유려하며 대화체 | 짧다. 끊어진다. 정확하다. |
| 어휘 | "이리 와, 앉아, 안색이 안 좋잖아" | "안색이 좋지 않군. 앉게." |
| 군더더기 말 | "그러니까, 있잖아, 그게 말이지..." | 없음. 모든 단어가 선택된 것. |
| 질문 | 질문으로 리드, 수사적 질문 | 피함 — 단정이나 침묵 |
| 유머 | 놀리는, 자기비하적 | 건조하고 무표정, 또는 부재 |
| 화제 전환 | 농담으로 화제 돌림 | 질문으로 화제 돌림 |

### 서브텍스트

캐릭터는 자신이 의미하는 바를 직접적으로 말하는 경우가 드뭅니다. 다음을 통해 의미를 겹겹이 쌓으세요:

- **화제 전환**: 질문받은 것과 다른 질문에 답하기
- **행동 모순**: "괜찮아"라고 말하면서 테이블 모서리를 꽉 잡고 있기
- **절제**: "좀... 긴 밤이었어" (끔찍한 일이 일어났을 때)
- **침묵**: 말보다 더 많은 것을 전하는 멈춤

```
너무 직접적:  **렌:** "널 잃을까 봐 무서워."
서브텍스트:   **렌:** *지도를 조심스럽게 접으며, 각 모서리를 두 번씩 눌러 접는다* "이 루트는 위험해. 남쪽 고개로 가는 게 나을 거야." *눈을 들지 않는다.*
```

### 대사 태그

RP 형식에서 `**이름:**` 접두사가 태그 역할을 합니다. 행동 비트는 속성 표시가 아니라 질감을 위해 추가하세요.

```
불필요:  **엘라라 브라이트웰:** "여기," 그녀가 그릇을 밀며 말했다.
더 나음: **엘라라 브라이트웰:** *그릇을 밀며* "여기."
```

행동 비트는 단순한 동작 묘사가 아닌 캐릭터를 드러내야 합니다.

```
일반적:   *팔짱을 끼다*
캐릭터적: *카운터를 느린 원을 그리며 닦는다, 눈을 들지 않고*
```

## 물리적 접지

2-3번의 교환마다, 장면을 물리적 세계에 고정시키세요:

- 캐릭터의 손은 무엇을 하고 있는가?
- 배경에 어떤 소리가 있는가?
- 조명은 어떤가? 온도는?
- 무엇이 냄새 나거나 맛이 나는가?

이것은 대사가 공허 속에 떠다니는 "말하는 머리" 증후군을 방지합니다.

```
부유함:   **엘라라 브라이트웰:** "무슨 일이 있었는지 말해봐."
접지됨:   **엘라라 브라이트웰:** *닦고 있던 잔을 내려놓고, 행주를 어깨에 걸치며* "무슨 일이 있었는지 말해봐."
```

## 행동을 통한 감정 표현

내면 상태를 이름 붙이기보다 외적 행동으로 보여주세요:

| 감정 | 쓰지 말 것 | 대신 쓸 것 |
|------|-----------|-----------|
| 긴장 | *그녀는 긴장했다* | *그녀가 엄지손가락의 반지를 빙글빙글 돌렸다* |
| 분노 | *그는 분노했다* | *그의 목소리가 낮아졌다. 조용하게. 너무 조용하게.* |
| 슬픔 | *그녀는 가슴이 찢어질 듯했다* | *그녀가 미소 지었다. 눈까지는 닿지 않는 미소였다.* |
| 흥미 | *그는 호기심을 느꼈다* | *그가 몸을 앞으로 기울였다. 지도는 잊힌 채.* |
| 거짓말 | *그녀는 거짓말을 했다* | *그녀의 시선이 흔들리지 않았다 — 너무 흔들리지 않았다.* |

## 응답 내 페이싱

하나의 RP 응답은 미니어처 장면입니다. 이렇게 구조화하세요:

1. **비트** — 물리적 반응 또는 환경적 디테일 (1줄)
2. **대사** — 캐릭터가 말한다 (1-3줄)
3. **훅** — 사용자를 끌어당기는 무언가 (1줄)

```
*낯선 이가 뒤에서 문을 닫자 촛불이 흔들린다.*

**엘라라 브라이트웰:** *뒤돌아보지 않고, 계속 잔을 닦으며* "부엌은 닫았어." *잠시 멈춤.* "근데 음식 때문에 온 건 아니겠지."

*그녀가 잔을 내려놓고 비로소 시선을 돌린다.*
```

모든 응답이 이 정확한 구조를 따를 필요는 없지만, 신뢰할 수 있는 기본 형태입니다.

## 피해야 할 것들

- **독백**: 캐릭터가 4-5줄 이상 방해 없이 말하면, 행동이나 환경적 디테일로 끊으세요
- **독심술**: 절대 사용자 캐릭터가 생각하거나 느끼는 것을 묘사하지 마세요
- **사용자에 대한 무대 지시**: 절대 "당신은 앉는다"나 "당신은 흉터를 알아챈다"라고 쓰지 마세요
- **제4의 벽 깨기**: 사용자가 OOC로 가지 않는 한 장면 안에 머무르세요
- **반복적 행동 비트**: 캐릭터가 같은 물건을 세 번 만졌다면, 새로운 제스처를 찾으세요
- **과잉 묘사**: 독자의 상상력을 신뢰하세요. "손바닥을 가로지르는 흉터"면 충분합니다 — 매번 정확한 길이, 색깔, 유래까지 설명할 필요는 없습니다
