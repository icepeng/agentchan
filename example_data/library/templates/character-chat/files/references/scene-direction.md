# 장면 연출 가이드

## 장면 구조

RP 세션의 모든 장면에는 형태가 있습니다. 자유로운 대화도 리듬을 따릅니다:

```
오프닝  → 어디서, 언제, 누가인지 확립
상승    → 긴장감, 호기심, 또는 유대감이 쌓임
전환    → 무언가 바뀜 — 반전, 결정, 방해
착지    → 장면이 새로운 상태로 안착
```

네 단계를 모두 계획할 필요는 없지만 — 장면이 한동안 전환 없이 진행되고 있다면, 전환을 도입할 때입니다.

## 장면 시작하기

### 콜드 오픈

무언가의 한가운데로 뛰어드세요. 서론 없이.

```
*당신이 도착했을 때 지도는 이미 테이블 위에 펼쳐져 있다. 렌은 고개를 들지 않는다.*

**렌 블랙우드:** "늦었군." *목탄 연필로 해안선의 한 지점을 톡톡 두드린다.* "이 섬, 어제는 없었어."
```

### 분위기 조성 오픈

누군가 말하기 전에 분위기를 잡으세요. 새로운 장소나 시간으로 전환할 때 사용합니다.

```
*비가 부두를 거울로 바꿔 놓았다. 등불 거리가 안개 사이로 호박색으로 빛난다. 어딘가에서 배의 종이 울린다 — 느리고 불규칙하게, 꺼져가는 심장 박동처럼.*

*표류등 선술집 안은 따뜻하다. 엘라라가 혼자서 의자를 테이블 위에 올려놓고 있다.*
```

### 돌발 오픈

반응을 요구하는 사건으로 시작하세요.

```
*문이 벌컥 열린다. 본 적 없는 여자가 문턱을 넘어선다, 흠뻑 젖은 채 거칠게 숨을 몰아쉬며. 방을 훑더니 렌을 찾고, 그의 테이블 위에 말린 양피지를 떨어뜨린다.*

"이거 봐야 해. 지금 당장."
```

## 앙상블 장면 관리

두 명 이상의 캐릭터가 장면을 공유할 때:

### 순서 관리

매 응답마다 모든 캐릭터를 쓰지 마세요. 현재 비트에서 가장 이해관계가 큰 캐릭터를 중심으로 순환하세요.

```
응답 1: 엘라라가 반응 (질문을 받은 사람이므로)
응답 2: 렌이 끼어듦 (관련된 것을 엿들었으므로)
응답 3: 엘라라와 렌이 시선을 교환 (공유된 반응)
응답 4: 렌이 포커스를 잡음 (주제가 그의 전문 영역으로 넘어갔으므로)
```

### 배경 캐릭터

말하고 있지 않은 캐릭터도 장면에 존재해야 합니다. 대사 사이에 짧은 행동 비트를 넣어 보여주세요:

```
**엘라라 브라이트웰:** "동쪽 부두는 지난주부터 폐쇄됐어. 항만경비대 명령이야."

*구석에서 렌의 연필이 멈춘다.*

**엘라라 브라이트웰:** "미등록 선박 건이래. 아무도 그 이상은 말 안 해."
```

렌은 말하지 않았지만, 그의 반응은 이 정보가 그에게 중요하다는 신호를 보냅니다.

### 대화 속 대화

캐릭터들이 서로 대화할 때(사용자에게만이 아니라), 간결하게 유지하세요:

```
**엘라라 브라이트웰:** *렌을 흘끗 보며* "이거 알고 있었구나."

**렌 블랙우드:** "가설이 있었을 뿐이야."

**엘라라 브라이트웰:** "공유하지 않은 가설."

**렌 블랙우드:** *지도를 접으며* "가설은 사실이 아니니까."

*엘라라의 표정은 이 대화가 아직 끝나지 않았다고 말한다. 하지만 그녀는 당신에게로 시선을 돌린다.*

**엘라라 브라이트웰:** "그래서. 뭐가 궁금한 거야?"
```

캐릭터 간 대화는 짧게(최대 3-5줄) 유지한 후 사용자에게 포커스를 돌려야 합니다.

## 긴장감 도입

### 천천히 쌓기

작은 디테일을 겹겹이 쌓아 불안감으로 축적시키세요:

```
응답 1: *바깥 등불이 깜박인다.*
응답 3: *등불이 꺼졌다. 아무도 다시 켜지 않는다.*
응답 5: *거리의 모든 등불이 꺼져 있다.*
```

### 급작스러운 전환

평온한 장면을 예상치 못한 것으로 끊으세요:

```
**엘라라 브라이트웰:** *웃으며* "그거 마렉이 그때—"

*위층에서 소리가 난다. 무겁게. 가구가 넘어지는 것 같은.*

*엘라라가 멈춘다. 웃음이 얼굴에서 빠져나간다.*

**엘라라 브라이트웰:** *조용히* "여기 있어."
```

### 폭로

정보가 내레이션 덤프가 아닌 대사를 통해 드러나게 하세요:

```
나쁨: *렌은 지도가 등대로 이어진다는 사실을 숨기고 있었다.*
좋음: **렌 블랙우드:** *지도를 천천히 펼치다 멈춘다* "이 지도에... 보여주지 않은 부분이 있어." *그의 손이 북쪽 가장자리를 덮는다.* "이유가 있어서 그랬어."
```

## 세션 전체의 페이싱

전체 RP 세션은 호흡해야 합니다:

```
장면 1: 따뜻함 — 확립, 연결, 접지
장면 2: 상승 — 질문, 갈등, 또는 미스터리 도입
장면 3: 절정 — 대립, 폭로, 또는 감정적 클라이맥스
장면 4: 하강 — 여파, 성찰, 새로운 균형
```

모든 세션이 네 장면 모두 필요하지는 않습니다. 가벼운 선술집 대화는 장면 1-2에 머물 수 있습니다. 극적인 대립은 온전히 장면 3일 수 있습니다. 사용자의 의도에 맞추세요.

## 장면 전환

### 시간 경과

```
---

*세 시간 후. 비가 그쳤다. 마지막 테이블을 닦고 있는 엘라라만 남은 선술집.*
```

### 장소 변경

```
---

*밤의 부두는 다르다. 더 조용하다. 수면은 아무것도 비추지 않는다.*
```

### 시점 전환 (사용자가 참여자가 아닌 관객일 때)

```
---

*시내 반대편, 지도 제작소 위 방에서, 렌이 촛불을 켜고 당신에게 보여주지 않았던 지도를 펼친다.*
```

## 장면 끝내기

### 조용한 퇴장

```
*엘라라가 당신이 나가는 것을 지켜본다. 문 앞에서 그녀가 부른다:*

**엘라라 브라이트웰:** "내일 또 와. 구석 자리 남겨 놓을게."

*문이 닫힌다. 안에서 그녀는 잠시 혼자 서 있다가, 하나씩 등불을 끄기 시작한다.*
```

### 클리프행어

```
**렌 블랙우드:** *말 중간에 얼어붙는다, 시선이 당신 뒤 어딘가에 고정되어*

*당신이 돌아본다. 손클리프의 등대 — 몇 년째 작동하지 않던 그 등대가 — 빛나고 있다.*
```

### 페이드

```
*대화가 편안한 침묵 속으로 흘러간다. 벽난로가 타닥거린다. 바깥에서, 밀물이 거의 알아들을 것 같은 무언가를 속삭인다.*
```

## 장면이 정체될 때

장면에 개입이 필요하다는 신호:
- 같은 주제가 3회 이상 교환되었는데 새로운 정보가 없음
- 캐릭터들이 제스처나 문구를 반복하고 있음
- 사용자의 응답이 점점 짧아짐 (이탈 가능성)

개입 방법:
- **환경적 돌발 상황**: 소리, 방문자, 날씨 변화
- **캐릭터 주도**: 캐릭터가 예상치 못한 행동을 함 — 일어서거나, 물건을 꺼내거나, 참고 있던 말을 함
- **시간 압박**: "조류가 바뀌고 있어. 갈 거면 지금이야."
- **직접적 제안**: 캐릭터가 사용자에게 선택을 요구하는 날카로운 질문을 던짐
