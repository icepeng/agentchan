# 달의 노래 전설 (The Moon Song)

## 개요

살레른의 민간 전설. 공식 종교(성 엘렌 신앙)와 별개로, **어부들 · 선원들 · 노년 거주민** 들이 믿거나 외는 옛 이야기. 음유시인들이 자주 부름.

## 전설의 줄기

> *세 바람이 한 밤에 마주칠 때 / 달의 어머니가 항구를 지켜주시네 / 그녀의 노래가 바다를 눕히고 / 길 잃은 배를 부두로 데려오네*

- **세 바람**: 북풍(시린 바람) + 남풍(짠 바람) + 서풍(안개 바람). 살레른 해안에서 교차
- **달의 어머니**: 이름 없는 신적 존재. 선원들은 "그분" (She)로만 지칭
- **노래**: 직접 부르는 게 아니라 **듣는** 것. 특정 조건에 들림

## 믿음의 세부

- 폭풍 · 해무 · 해상 위험에서 **달의 노래를 들은 자는 돌아온다**
- 노래를 듣는 조건 (어부 전승):
  1. 삼일월(月) 밤
  2. 세 바람이 교차 (기압 역전 시 자주 발생)
  3. 듣는 자가 **혼자**
- 노래의 내용은 **언어가 아님** — 떨림, 간격, 빛

## 과학적 해석 (셀렌 · 알라나)

- 세 바람 교차 시 기압차로 **저주파 울림** 발생 — 실제로 "노래처럼" 들리는 물리 현상
- 기압 변화 시점 = 폭풍 후 **안정기 시작** → "돌아올 수 있다" 는 경험이 전설화
- 셀렌 형제의 노트: "자연 법칙이 믿음을 만든다"

## 음유시인의 변주

이드리안은 이 노래를 **수십 가지 버전**으로 부름:

- 상업 버전: 축복 · 희망
- 구슬픈 버전: 전사자 · 실종자 추모
- 암호화 버전: **셀렌이 만든 것** — 첩보 시절 유산. 4~5개 단어가 특정 장소·시간·인물을 가리킴

## 셀렌의 암호 노래

- **정식 암호명**: `coded_whistle`
- 일반 청취자에겐 단순 민요로 들림
- 특정 프레이즈 = 특정 정보:
  - "세 바람이 늦어지네" = 정기 화물선 도착 지연
  - "달이 붉네" = 밀수 배가 들어옴
  - "매가 난다" = 다스렌 가문 관련 움직임
  - "등불이 꺼진다" = 고아원에서 실종 발생
- 해독법: 셀렌의 코드북 (수도원 도서관, trust +3 이상 접근 가능)

## 세 바람 교차 현상

- 실제로 살레른 해안에서 **월 2~3회** 발생 (기상 기록)
- 새벽 3~5시에 주로 발생
- 배가 항구로 돌아오기 좋은 시간대 — 과학적 근거가 있는 전승

## 플레이어가 전설을 활용할 때

- **어부 · 노인 NPC 대화 시** 화제로 쓰면 친밀감 +1 (단, 자주 쓰지 말 것)
- **이드리안에게 전설을 요청** — 분위기에 맞는 버전을 맞춤 공연
- **셀렌에게 암호 해독 요청** — trust +3 + 수도원 방문 3회 조건
