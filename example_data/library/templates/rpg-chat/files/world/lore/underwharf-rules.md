# 포세계 규율 (Underwharf Rules)

## 개요

- **공식 명칭 없음**. 현지인들은 "포세계" (아래 세상) 또는 "검은 부두"
- **공식적으로는 존재하지 않음** — 마샬 보고서에도 등장하지 않음
- **실제 영역**: 부두 뒤편 슬럼가 + 지하 수로 + 폐창고 3곳 (A/B/C)
- **현 두목**: 카엘렌 (왕국력 808년부터)

## 카엘렌 이전 (804~807)

- 기존 두목 3명 난립: "고양이" 산덴 / "말뚝" 브레인 / "자갈" 마르곤
- 주요 거래: 술 밀주 · 밀수 포도주 · 소규모 도박
- **인신매매 없었음** — 세 두목의 불문율

## 카엘렌 체제 (808~)

- 3 두목을 1년 안에 숙청. 방법은 아무도 모름 (공개 폭력은 최소)
- **"단일 영역, 단일 법칙"** 확립
- 주요 변화:
  - 밀수 규모 3배 확대 (북쪽 왕국 독주 중심)
  - 도박 공식화 (폐창고 A)
  - 장물 취급 전문화 (폐창고 B)
  - **폐창고 C** — 용도 비공개. 최근 활동 증가

## 포세계 규율 (구전)

### 공개된 것

1. **새 얼굴은 3일 관찰** — 낯선 자는 거래 직전까지 침묵 감시
2. **밤 거래 낮 숨김** — 낮에 포세계 언급 금지
3. **피 흘리기 금지** — 포세계 영역 내 살인 시 전원 연대 책임
4. **마샬과 충돌 회피** — 직접 공격 절대 금지

### 숨겨진 것 (카엘렌 개인 지시)

5. **다스렌 가문 건드리기 금지** — 부하들은 이유 모름
6. **비아 고아원에 기부 (익명)** — 카엘렌 개인 지시. 월 5 루아
7. **알라나 길드홀 정보는 "까치" 에게만** — 정보 집중화

## 인사법

- 공식: **"작은 잔에 큰 값"** — 거래 시작 인사
- 회원 구분: 왼쪽 손목 안쪽 **작은 타투** (조개 모양). 외부인에겐 보이지 않게
- 거절: 양 손바닥 보이기 (칼 없음의 표시)

## 거래 도구

- **중간 화폐** — 동으로 만든 조개 모양 토큰. 1 토큰 = 은화 1닢 (카엘렌 보증)
- **계약서 없음** — 구두·악수. 위반 시 카엘렌 재판 (공개 안 됨)
- **은매 (Hush Sale)** — 거래 내역 숨기는 고가 서비스. 추가 10% 수수료

## 세 폐창고

- **A (도박)** — 카드·주사위. 외부인도 가능. 경비 DC 13
- **B (장물)** — 전문 상품. 회원 전용. 카엘렌 집무공간 (19:00~03:00). 경비 DC 16
- **C (비공개)** — "중요한 것" 만. 접근 극히 제한. 경비 DC 18 + 까치 + 지하 통로로만
