# 옛 부두 말 (Old Harbor Tongue)

## 개요

살레른 항구의 선원 · 어부 · 노동자 사이에서 약 150년간 쓰이다 **점차 소멸** 중인 혼합 방언. 왕국 공용어(Theraval) + 남쪽 제도국 어휘 + 옛 공용어(Archaic Theraval) 잔재의 조합.

## 현재 사용자

- **노년 어부 · 노동자**: 60대 이상 대부분 이해, 일부만 구사
- **양조장 에스텔 (72세)** — 일상 언어로 사용
- **브라더 셀렌 (60세)** — 기도문에 간혹 사용, 왕실 첩보 시대 유산
- **다스렌 경 (68세)** — 이해하나 구사 안 함 (귀족은 피함)
- **30대 이하**: 대부분 이해 불가

## 주요 어휘 차이

| 옛 부두 말 | 공용어 | 의미 |
|---|---|---|
| *kalen* | 바람 | "한 숨의" |
| *meruvan* | 바다 | "엎어진 은" |
| *shunlai* | 안개 | "숨을 쉰다" |
| *dorren* | 부두 | "나무 뿌리" |
| *askani* | 저녁 | "불 밝힌 뒤" |
| *ferruka* | 달 | "둥근 어머니" |

## 인사 · 관용구

- **"달이 둥글기를"** (*ferruka kanan!*) — "안녕하세요" 의 정중한 표현
- **"바람을 거스르지 마시게"** (*kalen muru!*) — "조심하세요"
- **"엎어진 은이 잠잠한가?"** (*meruvan shun?*) — "별일 없나?"
- **"숨을 쉰다"** (*shunlai!*) — "안개가 짙다" 의 시적 표현

## 플레이어가 활용할 수 있는 경우

### 에스텔과의 대화

에스텔은 혼자 살아오면서 **옛 부두 말을 들어주는 사람이 없음**. 플레이어가 한 구절이라도 알면:

- 첫 반응: 놀람 → 기쁨
- 화술 어드밴티지 — 모든 대화에서
- 실종 목격담 (Act 2 witness_appears) 자발적 공유 확률 증가

단, **"공용어로 배운" 티** 가 나는 것은 오히려 역효과. 자연스럽게 한 구절 섞는 수준이 좋음.

### 셀렌과의 대화

- 옛 부두 말을 아는 자를 **동료로 인식**
- 코드북 접근 조건 일부 완화
- 일부 기도문·주문 의미를 설명해줌

### 다스렌 경과의 대화

- 의외의 깜짝. 귀족이 쓰지 않는 언어라 **신기함 + 호기심**
- 그러나 실질적 친밀감 상승은 거의 없음 (다스렌 경은 계급 의식이 강함)

## 플레이어의 학습 방법

- **shores / 항구 근처 노인과 대화** — 1~2 구절 자연 습득 (trust +1)
- **셀렌이 가르쳐줌** — trust +3 에서 선택 가능
- **에스텔이 가르쳐줌** — trust +3 에서 4~5 구절 습득

## 공통 문화 참조

옛 부두 말은 **달의 노래 전설**과 깊이 연결:

- "세 바람이 마주친다" 는 원래 *kalen kalen kalen meru* — "바람 바람 바람 마주"
- 음유시인 이드리안도 옛 부두 말 구절을 노래에 섞음 (원본 의미는 모른 채)
