# 살레른 밀수 경로 (Smuggler Routes)

## 개요

살레른의 공식 세관은 1번 부두 입구 한 곳. 나머지 2~5번 부두는 **자율 신고** 체제 — 즉 세관 공백이 크다. 이 공백이 밀수의 구조적 기반.

## 공식 경로

- **1번 부두**: 왕실 세관 검문 (평일 9:00 ~ 18:00, 주말 휴무)
- **관세**: 수입 상품 가치의 12% (왕실 재정 주요 원천)
- **면세 품목**: 종교용품 · 왕실 구매물 · 의료 약초

## 비공식 경로 (알려진)

### 1. 심야 하역 (Ship Night-Drop)
- 22:00 이후 4·5번 부두에 정박해 신속 하역
- 주로 독주 · 고가 염료 · 소금
- 카엘렌 조직 담당. 부하 10~15명 동원

### 2. 해안 동굴 (Coastal Caves)
- 살레른 북동쪽 2km, 조수 높을 때만 잠기는 동굴
- 소형 선박만 접근 가능 (나룻배 크기)
- **작은 짐 · 고가 밀수** 용: 보석, 독주, **인신매매**
- 이 경로는 `smugglers_route` 지도에 표시됨

### 3. 양조장 지하 통로 (Brewery Tunnel)
- `brewery` 지하 숙성실 뒤벽 → `undermarket` 폐창고 C 연결
- 길이 약 80m, 3인 통행 가능
- 할머니 에스텔은 존재를 알지만 침묵 (빚 때문에)
- **최근 사용 증가** — 실종 아동 사건과 연관 의심

## 외부 선박 현황

- **정기 화물선**: 월 4척 (공식 등록)
- **비정기**: 월 2~6척 — 일부는 밀수 대응
- **실제 미등록 선박**: 월 1~2척 (조선사 불명, 국적 불명)
- 이 미등록 선박이 **인신매매 운반선** 으로 의심됨

## 관습 · 암호

### 부두 노동자 기호 (벽의 흔적)

- **붉은 X**: "오늘 위험"
- **흰 O**: "안전, 정상 운영"
- **검은 V**: "밤에 다시"
- **붉은 V**: "절대 금지"

이 표식은 1번 부두 입구 옆 나무 게시판에서 매일 교체.

## 세관 부패 (의심)

- 세관 인원 4명 중 2명이 알라나 길드 연줄 (비공식)
- 길드가 내는 "협력비" → 통관 간소화
- 테렌 마샬도 이를 알고 있으나 증거 부족

## 최근 변화 (Act 1~2)

- 심야 하역 빈도 증가 (주 3회 → 주 5회)
- 해안 동굴 경로 사용 징후 (새 발자국)
- 양조장 지하 통로 사용 — 에스텔이 목격

## 플레이어의 추적 방법

- **부두 노동자와 친해지기**: 표식 의미 파악 (trust +1 로 충분)
- **셀렌의 코드북**: 음유시인의 노래로 밀수 타이밍 파악
- **일하 관찰**: 그녀가 부두 뒤 창고를 감시하는 위치 포착
- **양조장 에스텔**: 직접 목격담 (Act 2 witness_appears 트리거)
