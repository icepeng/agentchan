# 달의 눈 여관 (The Moon's Eye Inn)

중앙 광장에 면한 여관. 플레이어의 베이스캠프. 1층 술집, 2층 방 4개.

## 풍경

살레른 중앙 광장 북쪽 측면, 나무+돌 혼합 2층 건물. 간판에 **초승달 + 눈 문양** — 오래되어 색이 바랜 상태.

- **1층 술집**: 중앙에 벽난로, 의자 20석. 카운터 뒤에 여관주 **타렌** (50대 미망인)
- **2층 객실**: 4개. #1, #2 (싱글), #3 (더블), #4 (스위트 · 가장 큰 방). 플레이어는 #2 에 숙박
- **뒷마당**: 작은 우물 + 빨래줄 + 닭장 5마리
- **지하**: 저장고 (에일 · 와인)

## 분위기

- **낮**: 손님 5~10명. 부두 노동자 점심, 지역 상인 회동
- **저녁**: 20~30명. 음유시인 (이드리안) 자주 방문, 이야기판
- **밤**: 손님 줄어들다가 23:00 소등. 투숙객만 남음
- **벽난로 옆 자리**: **가장 조용한 이야기 장소**. 중요한 대화가 자주 일어남

## 주요 인물

- **타렌 (여관주)** — 단역, 50대. 플레이어에게 호의적. 정보 좋아함
- **이드리안** — 저녁마다 1~2시간 공연. 공짜 식사가 숙박료
- **단골 손님** — 부두 노동자 3~4명, 지역 상인 2명

## 서비스

- **숙박**: 하루 은화 1닢. 아침 포함 (귀리 죽 + 훈제 생선 + 차)
- **식사**: 스튜 / 빵 / 에일 (각 1~3 동전)
- **목욕물**: 은화 1닢 (드묾 · 사전 예약)

## 흔적 · 단서

- **벽난로 옆 게시판**: 거래·구인·분실물 공지. 가끔 중요한 단서
- **타렌의 잡담**: 주요 손님 동향 파악 가능 (trust +1 로 충분)
- **2층 복도**: 새벽 종종 낯선 발자국 소리 — 플레이어가 일어나 있다면 목격 가능 (특정 비트 트리거)

## 휴식 규칙

- **짧은 휴식 (1시간, 은화 0)**: 벽난로 앞 의자. MP +1, HP +2. 1시간 경과
- **긴 휴식 (숙소 밤샘)**: MP 전체 회복, HP 전체 회복. 8시간 경과 (일반적으로 22:00 ~ 06:00)
- **동료 대화 트리거**: 긴 휴식 시 조건 만족하는 `bond_moments` 자동 발화 가능

## 이동

- 중앙 허브 — 5개 장소와 연결 (`pier`, `guild_hall`, `monastery`, `undermarket`, `brewery`)

## 시간 / 상태

- `safe: always` — 타렌이 폭력 용납 안 함. 여관 내부 전투 시 즉시 쫓겨남
- `faction: none` — 중립
