# 포세계 골목 (Undermarket Alleys)

카엘렌의 영역. 밤에만 열리는 장터, 낮에는 빈 골목.

## 풍경

부둣가 뒤편 구역 — 좁은 골목 네트워크 + 지하 수로 + 폐창고. **공식적으로는 존재하지 않는** 장터. 낮에는 평범한 슬럼처럼 보임: 빨래줄, 울부짖는 아이, 빈 수레.

밤이 되면 다른 곳. **20:00 이후** 각 폐창고의 문이 열리고, 거래상·도박판·정보 교환·장물 취급이 시작됨. 21:00 ~ 02:00 가 최고조.

- **중앙 광장** (작은): 야시장. 천막 12~15개
- **지하 수로 입구 3곳**: 카엘렌 부하 전용
- **폐창고 A · B · C**: A = 도박, B = 장물, C = **인신매매 (은폐, 저택 지하실과 통로 연결)**

## 분위기

- **낮**: 조용. 빨래 말리는 소리, 외부인에게 **무관심**. 민간 경비 없음 (경비가 오지 않는 구역)
- **밤**: 활기. 횃불과 기름등. 낯선 얼굴이 오면 **세 명이 동시에 쳐다봄**
- **새벽**: 천막 철거. 낮 모습 복귀

## 주요 인물

- **카엘렌** — 폐창고 B 안쪽 집무 공간 (19:00 ~ 03:00). 낮에는 여기 없음
- **부하장 "까치"** (단역) — 키 큼, 왼쪽 팔 문신. 카엘렌 직속. 경비 DC 18
- **일반 부하** × 다수 — 경비 DC 13~15
- **단골 거래상** — 정보 판매 가능 (화술 DC 14 + 은화 2닢)

## 흔적 · 단서

- **폐창고 C 지하 통로**: 다스렌 저택 지하실과 연결 (Act 2 핵심 단서)
- **밤 골목 흔적**: 아이 발자국 (관찰 DC 14) — 실종 아동의 이동 경로
- **장물 선반**: `smugglers_route` 지도 단편 (은밀 민첩 DC 16 or 화술 DC 17 거래)

## 진입 방법

- **낮 방문**: 자유. 정보 거의 없음
- **밤 방문**: 가능하나 **민첩 디스어드밴티지** (어둠). 리우 동반 시 어드밴티지 상쇄
- **거래 참여**: 화술 DC 14 + 뇌물 가능. 시간이 걸림
- **은밀 조사**: 민첩 DC 16 (연기탄 · 리우 · veil 주문 등 보조 권장)

## 이동

- `pier` (8분), `inn` (12분), `brewery` (6분 · 지하 통로 경유 가능)

## 시간 / 상태

- `faction: underwharf` — 카엘렌 영역. `trust:kaelen` 값이 분위기 결정
