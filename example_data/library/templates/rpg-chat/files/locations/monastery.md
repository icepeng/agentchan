# 성 엘렌 수도원 (St. Elen's Monastery)

언덕 위 작은 수도원. 셀렌 형제를 포함해 수도승 다섯 명.

## 풍경

살레른에서 가장 높은 지점. 도시에서 도보 20분. 흰 석회암 3층 건물 + 예배당 + **약초 정원**.

- **예배당**: 수용 60명. 성 엘렌 석상, 촛대 9개 (9성인 숭배)
- **수도원 본관 1층**: 식당 + 부엌
- **2층**: 도서관 (고문서 200여 권) + 수도승 방 5개
- **3층**: 원장 거처 (접근 제한)
- **약초 정원**: 셀렌 형제 담당. 12종 약초 + 치유용 허브

## 분위기

- **새벽~오전**: 기도 · 노동. 예배당 종소리 6:00 · 9:00 · 12:00
- **오후**: 셀렌 형제가 약초 정원에서 일함. 가끔 허밍이 들림
- **저녁**: 공동 식사 (18:00) + 야간 기도 (21:00)
- **밤**: 완전 적막. 수도원은 **피난처 전통** — 도피 중인 자를 하룻밤 받아줌

## 주요 인물

- **셀렌 형제** — 주로 약초 정원 (10:00 ~ 17:00), 도서관 (그 외)
- **원장 (60대)** — 3층. 플레이어가 만날 기회는 매우 드묾
- **수도승 3명** — 단역. 식사·노동·기도

## 흔적 · 단서

- **도서관 고문서**: **옛 왕국 코드북** (셀렌 개인 서가 · `coded_whistle` 해독 시 접근 가능)
- **약초 정원**: `purify` 주문 재료 공유 가능 (셀렌 trust +2 부터)
- **예배당 뒤편**: 비아가 익명으로 남긴 기부 흔적 (`trust:via` 힌트)

## 진입 방법

- **정식 방문**: 자유. 예배당·식당은 누구나
- **도서관**: 원장 허가 또는 셀렌 trust +3 (셀렌이 안내)
- **3층 원장 거처**: 거의 불가능. 엔딩 근처 "수도원 피난" 시나리오에서만

## 이동

- `manor` (18분 · 언덕 서쪽 내려와 동쪽으로)
- `inn` (20분 · 언덕 내려오기)

## 시간 / 상태

- `safe: always` — 절대적 안전. 수도원 관습상 **폭력 금지**
- `faction: none` — 중립. 모든 세력이 존중

## 특별 규칙

- **수도원 내부 전투 금지** — 어떤 NPC도 예배당/본관에서 싸움 시작 안 함. 플레이어가 먼저 시작하면 즉시 쫓겨남 + 모든 NPC trust -2
- **피난처** — 플레이어가 심각한 위기 시 이곳으로 도피 가능 (추격자 중단)
- **단기 휴식**: 1시간 MP +1. 장기 휴식 (밤샘): MP 전체 회복 + HP 5 회복
