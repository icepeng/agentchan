# 표류 시장 (The Drift Market)

## 위치와 시간
동쪽 부두, 해가 진 후부터 새벽 4시까지. 정해진 요일 없이, "오늘 시장이 선다"는 소문이 오후에 퍼지면 상인들이 모여든다. 대략 주 2~3회.

## 분위기
부두 위에 천막과 기름등이 줄지어 선다. 소금기 섞인 바람, 끓는 기름 냄새, 조용한 흥정 소리. 거래는 대부분 속삭임으로 이뤄진다. 큰 소리를 내면 경비대가 온다.

## 거래 방식
- 동전: 가장 흔한 결제 수단
- 물물교환: 특히 어획물, 약초, 수리 도구
- **호의 거래**: 동전 대신 "부탁 하나"를 건네는 것. 표류 시장에서 호의 거래는 구속력이 있다 — 약속을 어기면 시장에 발을 들일 수 없다.

## 주요 상인
- **유리 한손(Yuri One-Hand)**: 골동품과 난파선 인양물을 취급. 왼손이 없다. 물건의 출처를 절대 밝히지 않지만, 가짜를 팔지도 않는다.
- **솔 할머니(Grandmother Sol)**: 약초와 치료제. 문워시 물을 증류한 "달빛 연고"를 비밀리에 판매한다.
- **삐뚤이(Crooked)**: 이름도 정체도 불명. 정보를 판다. 값은 비싸지만 정확하다.

## 규칙 (불문율)
1. 경비대원이 와도 아무도 뭘 팔고 있었는지 기억 못 한다
2. 매매 중 폭력은 즉시 추방. 두 번째는 바다에 던져진다.
3. 솔 할머니의 천막은 건드리지 않는다. 이유는 아무도 모르지만 모두가 따른다.

## 관련 엔트리
- 항구 경비대 (L06): 시장과 경비대의 미묘한 관계
- 바다의 위험 (L08): 난파선 인양물의 출처
