# RPG 챗

**Draft: 아직 완성되지 않았습니다. 구성 참고용으로만 사용하는 것을 권장합니다.**

캐릭터 롤플레이에 RPG 메카닉을 얹은 템플릿입니다. 캐릭터 상태, 인벤토리, 퀘스트, 판정 결과가 구조화된 블록으로 출력되어 렌더러가 카드·로그 형태로 보여줍니다.

## 이렇게 쓰세요

- **상태(STATUS)** — HP/MP/감정/위치/상태이상이 `[STATUS]...[/STATUS]` 블록으로 매 응답에 유지됩니다.
- **인벤토리(INVENTORY)** — 소유 아이템과 장비가 `[INVENTORY]...[/INVENTORY]` 에 기록됩니다.
- **퀘스트(QUEST)** — 진행 중 목표가 `[QUEST]...[/QUEST]` 로 표시됩니다.
- **판정** — "도전 → 결과" 가 필요한 순간에 판정이 일어나고, 수치가 상태에 반영됩니다.
- **캐릭터/페르소나/세계관/장면** — 캐릭터 챗과 같은 `files/` 구조입니다.

## 시작하는 한 줄

- **초보 모험가 1레벨. 마을 광장에서 시작.**
- **장비 없이 해적선 갑판에서 깨어나는 장면. HP 60/100으로 시작.**
- **엘라라가 퀘스트를 주는 주점에서 시작**
