# 인터랙티브 챗

**Draft: 아직 완성되지 않았습니다. 구성 참고용으로만 사용하는 것을 권장합니다.**

AI가 캐릭터 응답 뒤에 **항상 3개의 선택지**를 붙입니다. 선택지를 클릭하면 그대로 다음 입력이 되고, 물론 직접 자유 입력도 할 수 있어요. 가벼운 게임북 느낌으로 장면을 넘길 때 좋습니다.

## 이렇게 쓰세요

- **캐릭터/페르소나/세계관** — 캐릭터 챗과 동일한 구조입니다. `files/characters/`, `files/personas/`, `files/world/`.
- **선택지** — 매 응답의 마지막에 `[CHOICES]...[/CHOICES]` 블록이 붙습니다. 렌더러가 이를 버튼으로 표시합니다.
- **장면 기록** — `files/scenes/scene.md` 에 append 됩니다.

선택지는 보통 "적극적 / 탐색적 / 소극적" 세 방향으로 균형 있게 제시됩니다. 의도를 강제하지 않으니 언제든 자유 입력으로 넘어가도 됩니다.

## 시작하는 한 줄

- **길에서 처음 만나는 낯선 사람과의 조우로 시작**
- **엘라라의 주점에서 의뢰를 고르는 장면**
- **선택지는 3개 유지. 진중한 톤으로 시작해줘**

## 더 알아보기

- 선택지 UI가 부담스러우면 **Character Chat** 으로 전환하세요 — 같은 캐릭터를 쓸 수 있습니다.
