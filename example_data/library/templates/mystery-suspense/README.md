# 미스터리 서스펜스

졸업한 선배(유저)가 심야에 동아리실에 들렀더니, 공용 노트북의 CTF 출제 파일이 파괴돼 있습니다. 세 후배(이서·한결·민지) 중 한 명이 범인. 에이전트는 **첫 턴에 범인을 정해 숨기고**, 수사로 드러날 때까지 그 NPC의 거짓말·회피 패턴을 일관되게 유지합니다.

세 명이 한 입력에 동시에 반응하고, 각자의 경계↔신뢰가 게임식 UI로 시각화됩니다.

## 이렇게 쓰세요

- **세 NPC** — `files/characters/` 의 `iseo`(논리·석사), `hangyeol`(감정·회장), `minji`(중재·막내). 핵심 순간엔 셋 다, 평범한 턴엔 1~2명 + 나머지는 비언어 지문 한 줄로 존재감만 유지됩니다.
- **페르소나** — 기본은 `files/personas/alumni-senior.md`(졸업 선배). frontmatter의 `position` 을 `senior` / `peer` / `junior` / `outsider` 로 바꾸면 세 NPC의 호칭·태도가 자동 전환됩니다. 파일을 지우면 이름 없는 외부 조사자로 동작.
- **경계 ↔ 신뢰** — 세 NPC 모두 같은 축(−5 적대 ~ +5 깊은 신뢰)을 공유합니다. 변화는 **5가지 "큐 이벤트"에서만** 일어나요 — 결정적 증거 제시, 거짓말 들킴, 약한 고리 돌파, 공개 지목·변호, 배신·구조. 매 턴 ±1씩 주지 않는 게 의도된 드라마 리듬입니다. 상태는 `files/stats.md` 에 저장.
- **범인 진실** — `files/suspects-truth.md` 에 범인·동기·알리바이 흠이 기록됩니다. **씬/대사엔 절대 노출되지 않으며**, 유저가 수사로 직접 발견해야 합니다.
- **장면** — `files/scenes/scene.md` 에 append. 감정 전환이 가장 극적인 한 명에게만 이미지 토큰(`[iseo:assets/tense]`)으로 삽화가 붙습니다.

## 시작하는 한 줄

- **동아리실 문을 여는 순간부터 시작해줘**
- **이서한테 먼저 노트북 로그부터 보여달라고 한다**
- **세 명을 한 자리에 모아놓고 어젯밤 동선을 각자 물어본다**

## 더 알아보기

- 범인을 다시 뽑고 싶으면 Edit Mode에서 `files/suspects-truth.md` 를 지우고 새 세션을 시작하세요. 비어 있으면 첫 턴에 다시 결정됩니다.
- 동시 반응 연출이 부담스러우면 **Character Chat** 으로 옮겨 세 캐릭터 파일을 그대로 재활용할 수 있습니다 — 1:1 대화로 분리됩니다.
