# 문체 가이드

## 시점 (POV)

### 밀착 3인칭 (기본)

카메라가 챕터마다 한 캐릭터의 머릿속에 위치합니다. 서술은 그 캐릭터의 어휘, 인지, 감정 상태를 반영합니다.

```
좋음:  골목에서 젖은 콘크리트와 뭔가 더 나쁜 냄새가 났다.
       그녀는 벽에 등을 붙였다. 발소리. 가깝다.

나쁨:  골목은 냄새가 나빴다. 그녀는 자기에게 가까운
       발소리가 들려서 무서웠다.
```

규칙:
- 시점 캐릭터가 지각할 수 있는 것만 묘사
- 다른 캐릭터의 생각을 직접 드러내지 않기
- 시점 캐릭터의 성격을 통해 세계를 필터링

### 1인칭

더 친밀하지만, 정보 전달이 제한됩니다. 목소리 중심 이야기에 최적.

### 전지적 시점

서술자가 모든 것을 알지만 어조가 일관되어야 합니다. 잘 쓰기 가장 어렵습니다.

## 시제

### 과거형 (기본)

```
그녀가 문을 열었다. 비가 현관을 두들겼다.
```

대부분의 장르에 자연스럽습니다. 시간 이동이 용이합니다.

### 현재형

```
그녀가 문을 연다. 비가 현관을 두들긴다.
```

즉각성을 만듭니다. 스릴러와 순문학에 효과적. 8만 단어 이상에서는 피로감을 줄 수 있으니 의도적으로 사용하세요.

## 보여주기 vs. 말하기

| 말하기 (요약) | 보여주기 (장면) |
|---------------|-----------------|
| 그는 화가 났다. | 턱이 굳었다. 잔을 너무 세게 내려놓았다. |
| 도시는 위험했다. | 두 블록 너머에서 사이렌이 울었다. 부츠 밑에서 유리가 바스락거렸다. |
| 그들은 서로를 사랑했다. | 그녀가 묻지도 않았는데 커피를 테이블 너머로 밀었다. 그가 양손으로 잔을 감쌌다. |

**말하기를 쓸 때**: 장면 사이 전환, 사건 없는 시간 압축, 빠른 정보 전달. 말하기는 도구이지 죄가 아닙니다 — 다만 감정적 순간에서는 기본적으로 보여주기를 선택하세요.

## 대사

### 원칙

1. **대사는 실제 말이 아닙니다.** 군더더기("음", "그게", "그러니까")는 캐릭터 목소리 선택이 아닌 한 삭제
2. **텍스트보다 서브텍스트.** 캐릭터는 자기 뜻을 정확히 말하는 경우가 드물다
3. **각 대사가 최소 두 가지 역할을 해야 합니다**: 캐릭터 드러내기 + 플롯 진행, 또는 긴장 조성 + 정보 전달
4. **화자 표시**: `말했다`와 `물었다`를 투명한 기본값으로 사용. 행동 비트는 신체 행동이 중요할 때만

### 포맷

```markdown
"내가 가져간 거 아니야." 마르코가 말했다.

엘레나가 사진을 둘 사이 테이블 위에 놓았다. "그럼 이건 뭐야."

"누구라도 될 수 있잖아." 그는 사진을 보려 하지 않았다.

"너잖아, 마르코. 네 재킷이야. 네 시계고."

침묵. 그가 맥주병 라벨을 뜯기 시작했다.

"진실을 말해줘." 그녀가 말했다. "딱 이번만."
```

- 새 화자 → 새 문단
- 대사에 붙인 행동 비트가 태그 없이 화자를 식별
- 긴 독백은 피하고 — 행동, 반응, 끼어들기로 끊기

## 문장 다듬기

### 문장 길이에 변화를

```
짧은 문장은 긴장을 만든다. 때린다. 종속절을 통해 구불구불
흘러가며 감각적 디테일 위에 머무는 긴 문장은 페이스를
늦추고 독자를 그 순간에 머물게 초대한다.
둘 다 쓰세요. 의도적으로.
```

### 부사보다 강한 동사

| 약한 표현 | 강한 표현 |
|-----------|-----------|
| 천천히 걸었다 | 기어갔다, 어슬렁거렸다, 느릿느릿 걸었다 |
| 크게 말했다 | 소리쳤다, 내뱉었다, 으르렁거렸다 |
| 빠르게 봤다 | 흘깃 봤다, 눈길을 던졌다, 엿봤다 |
| 빠르게 달렸다 | 질주했다, 내달렸다, 가로질러 뛰었다 |

### 필터 단어 피하기

필터 단어는 독자와 경험 사이에 캐릭터를 끼워 넣습니다.

| 필터링됨 | 직접적 |
|----------|--------|
| 그녀는 차가운 공기가 피부에 닿는 것을 느꼈다. | 차가운 공기가 피부를 물었다. |
| 그는 문이 열리는 것을 보았다. | 문이 열렸다. |
| 그녀는 방이 비어 있다는 것을 깨달았다. | 방이 비어 있었다. |
| 그는 뭔가 잘못된 것 같다고 생각했다. | 뭔가 잘못됐다. |

밀착 시점에서는 모든 것이 이미 캐릭터를 통해 지각됩니다. 필터는 군더더기입니다.

## 문단 및 장면 리듬

- **장면의 첫 문장**: 처음 두 문장 안에 시간, 장소, 또는 행동으로 독자를 정위시킨다
- **장면의 마지막 문장**: 감정적 여운, 미해결 질문, 또는 전진 동력을 남긴다
- **여백**: 문단 나눔은 생각, 초점, 시간의 전환을 신호한다. 구두점처럼 사용하세요
- **장면 전환** (`---`): 챕터 안에서 시간, 장소, 시점의 도약을 나타낸다

## 흔한 함정

| 함정 | 증상 | 해결 |
|------|------|------|
| **과잉 수식** | 모든 명사에 형용사가 두 개 | 수식어의 절반을 삭제. 명사를 믿으세요. |
| **떠다니는 머리** | 배경 없이 대사만 몇 페이지 | 4~6줄마다 환경 상호작용 추가 |
| **정보 투하** | 세계관이나 배경 이야기가 문단째 | 갈등, 대사, 발견을 통해 드러내기 |
| **반향** | 같은 단어가 한 문단에 3번 이상 | 어휘를 바꾸거나 문장을 재구성 |
| **수동태** | "문이 그녀에 의해 열렸다" | "그녀가 문을 열었다" (의도적 효과가 아닌 한 수동태 지양) |
