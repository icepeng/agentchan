## 분할 레이아웃

agentchan의 화면은 세 영역으로 구성됩니다.

- **왼쪽 패널**: 렌더링된 출력 영역. 프로젝트의 파일들이 렌더러를 통해 HTML로 변환된 결과를 표시합니다.
- **오른쪽 패널**: 에이전트 채팅 패널. 에이전트와의 대화가 표시됩니다. 접기/펼치기가 가능합니다.
- **하단**: 메시지 입력 영역. 에이전트에게 보낼 메시지를 작성합니다.

## 사이드바

사이드바에서는 다음 항목에 접근할 수 있습니다.

- **프로젝트 목록**: 생성된 프로젝트들이 나열됩니다. 클릭하면 해당 프로젝트로 전환됩니다.
- **템플릿 탭**: 새 프로젝트를 만들기 위한 템플릿 목록입니다.
- **설정**: AI 프로바이더 설정 등 앱 전체 설정으로 이동합니다.

## 뷰 모드

agentchan에는 두 가지 뷰 모드가 있습니다. AgentPanel 헤더에서 토글할 수 있습니다.

- **Chat 모드**: 에이전트와 대화하는 기본 모드입니다. 메시지를 주고받으며 에이전트가 파일을 수정합니다.
- **Edit 모드**: 프로젝트의 파일을 직접 편집하는 모드입니다. 파일 트리와 코드 에디터가 표시되어 SYSTEM.md, skills/, renderer.ts, files/ 등을 직접 수정할 수 있습니다.

## 렌더러 액션

렌더링된 HTML 내에는 클릭 가능한 인터랙티브 요소가 포함될 수 있습니다.

- **`data-action="send"`**: 클릭 시 해당 텍스트를 즉시 에이전트에게 전송합니다.
- **`data-action="fill"`**: 클릭 시 해당 텍스트를 입력창에 채웁니다. 전송하기 전에 내용을 수정할 수 있습니다.

Interactive Chat 템플릿의 선택지 버튼이 대표적인 예시입니다. 렌더러가 출력한 선택지를 클릭하면 해당 내용이 에이전트에게 자동으로 전달됩니다.
