## 다운로드

[GitHub Releases](https://github.com/anthropics/agentchan/releases) 페이지에서 OS에 맞는 실행 파일을 다운로드하세요.

- **Windows**: `.exe` 파일
- **macOS**: macOS용 바이너리
- **Linux**: Linux용 바이너리

실행 파일 하나로 동작하며, 별도의 설치 과정이 필요 없습니다. 다운로드 후 바로 실행할 수 있습니다.

## 첫 실행

실행 파일을 실행하면 브라우저에 agentchan 인터페이스가 열립니다. 사이드바에 **프로젝트 목록**과 **템플릿 탭**이 표시됩니다.

## AI 프로바이더 설정

대화를 시작하기 전에 AI 프로바이더를 설정해야 합니다.

1. 사이드바 하단의 **설정(Settings)** 페이지로 이동합니다.
2. 사용할 AI 프로바이더의 API 키를 입력합니다.

지원하는 프로바이더:

- **Anthropic** (Claude)
- **OpenAI**
- **OpenAI 호환 커스텀 프로바이더** — OpenAI API 형식을 지원하는 서비스라면 엔드포인트 URL과 API 키를 지정하여 사용할 수 있습니다.

## 첫 프로젝트 만들기

1. 사이드바의 **템플릿 탭**을 클릭합니다.
2. 원하는 템플릿을 선택합니다. 처음이라면 **Character Chat** 템플릿을 추천합니다.
3. 프로젝트 이름을 입력하고 생성 버튼을 클릭합니다.

프로젝트가 생성되면 자동으로 열립니다.

## 첫 대화

화면 하단의 입력창에 메시지를 입력하고 전송하세요. 에이전트가 SSE 스트리밍으로 실시간 응답합니다.

응답이 완료되면 왼쪽 패널의 **렌더러 출력**이 자동으로 갱신됩니다. 렌더러는 프로젝트의 파일들을 HTML로 변환하여 시각적으로 보여줍니다.
