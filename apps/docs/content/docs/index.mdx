agentchan은 **AI 크리에이티브 에이전트 데스크톱 앱**입니다. 캐릭터 롤플레이, 인터랙티브 소설, 장편 집필 등 창작 프로젝트를 AI 에이전트와 함께 진행할 수 있습니다.

## 핵심 특징

### 프로젝트 기반 작업

agentchan에서는 모든 창작 활동이 **프로젝트** 단위로 이루어집니다. 각 프로젝트는 독립된 작업 공간을 갖고 있어 캐릭터 정의, 세계관, 장면, 시스템 프롬프트 등을 체계적으로 관리할 수 있습니다.

### 2-Concept 아키텍처

agentchan의 설계는 두 가지 핵심 개념으로 구성됩니다:

- **SYSTEM.md** — 프로젝트 행동의 단일 원천. 에이전트가 어떻게 행동할지를 정의하는 시스템 프롬프트입니다.
- **스킬(Skill)** — 온디맨드 절차. 필요할 때만 활성화되어 에이전트에게 특정 작업 방법을 알려줍니다.

이 두 가지 외의 모든 것은 `files/` 디렉토리에 자유롭게 저장되는 **파일**입니다.

### 렌더러 시스템

각 프로젝트에는 **렌더러**(`renderer.ts`)가 있습니다. 렌더러는 프로젝트의 파일들을 시각적인 HTML로 변환하는 TypeScript 함수입니다. 캐릭터 채팅의 말풍선, 소설의 원고 뷰어 등 프로젝트마다 고유한 시각적 표현이 가능합니다.

### 에이전트 도구

AI 에이전트는 파일을 읽고, 쓰고, 편집하고, 검색하는 도구를 사용합니다. 에이전트의 작업 결과는 실시간으로 렌더링되어 왼쪽 패널에 표시됩니다.

## 인터페이스 구조

agentchan의 화면은 **분할 레이아웃**으로 구성됩니다:

- **왼쪽 패널** — 렌더링된 출력 (에이전트가 작성한 파일의 시각적 결과)
- **오른쪽 패널** — 에이전트 채팅 (AI와의 대화, 접기/펼치기 가능)
- **하단** — 메시지 입력
