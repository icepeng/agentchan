## 템플릿에서 생성

1. 사이드바의 **템플릿 탭**을 클릭합니다.
2. 원하는 템플릿을 선택합니다.
3. 프로젝트 이름을 입력합니다.
4. 생성 버튼을 클릭합니다.

## 복사되는 파일

템플릿에서 프로젝트를 생성하면 다음 파일들이 모두 새 프로젝트로 복사됩니다.

- **SYSTEM.md** — 시스템 프롬프트
- **renderer.ts** — 렌더러
- **skills/** — 스킬 파일들
- **files/** — 작업 파일들 (캐릭터, 세계관 등 초기 콘텐츠)

## 프로젝트 이름 규칙

프로젝트 이름은 한국어 그대로 사용할 수 있습니다. 내부적으로 폴더명은 slug 형태로 변환됩니다.

- 공백은 하이픈(`-`)으로 변환
- ASCII 문자는 소문자로 변환
- 한국어는 그대로 유지

예: "나의 첫 프로젝트" → `나의-첫-프로젝트`

## 생성 후

프로젝트가 생성되면 자동으로 열리고, 바로 채팅을 시작할 수 있습니다.
