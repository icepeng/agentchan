## AI 프로바이더 설정

agentchan은 외부 AI 프로바이더의 API를 사용하여 에이전트를 구동합니다. 설정 페이지에서 프로바이더와 모델을 구성합니다.

### 기본 프로바이더

기본적으로 다음 프로바이더를 지원합니다.

- **Anthropic** — Claude 모델 시리즈
- **OpenAI** — GPT 모델 시리즈

### API 키 입력

설정 페이지에서 사용할 프로바이더의 API 키를 입력합니다. API 키는 각 프로바이더의 공식 사이트에서 발급받을 수 있습니다.

### 모델 선택

프로바이더별로 사용 가능한 모델 목록이 표시됩니다. 원하는 모델을 선택하여 에이전트가 사용할 AI 모델을 지정합니다.

## 커스텀 프로바이더

OpenAI 호환 API 엔드포인트를 커스텀 프로바이더로 추가할 수 있습니다. 로컬 LLM이나 다른 호환 서비스를 연결할 때 사용합니다.

### 설정 항목

커스텀 프로바이더를 추가할 때 다음 항목을 입력합니다.

- **Base URL** — API 엔드포인트 주소 (예: `http://localhost:11434/v1`)
- **API 키** — 인증에 사용할 API 키 (로컬 서비스의 경우 빈 값 또는 임의의 값)
- **모델 이름** — 사용할 모델의 식별자

### 활용 예시

- **Ollama** — 로컬에서 실행 중인 Ollama 서버에 연결하여 오픈소스 모델을 사용합니다.
- **LM Studio** — LM Studio의 로컬 서버를 프로바이더로 등록합니다.
- **기타 호환 서비스** — OpenAI 호환 API를 제공하는 모든 서비스를 연결할 수 있습니다.
