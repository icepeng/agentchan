## 스킬이란?

스킬은 `skills/` 디렉토리의 `SKILL.md` 파일로 정의된 온디맨드 절차입니다. 시스템 프롬프트에는 카탈로그(이름+설명)만 포함되고, 스킬 본문은 활성화 시에만 에이전트에게 전달됩니다.

이렇게 하면 토큰을 절약하면서도 필요할 때 상세한 지시를 제공할 수 있습니다. 모든 절차를 시스템 프롬프트에 넣으면 토큰이 낭비되지만, 스킬로 분리하면 필요한 순간에만 로드됩니다.

## 활성화 방법

스킬을 활성화하는 두 가지 방법이 있습니다.

### 도구 호출

에이전트가 스스로 판단하여 `activate_skill` 도구를 호출합니다. 시스템 프롬프트의 카탈로그에서 현재 상황에 적합한 스킬을 선택하여 활성화합니다. 사용자가 별도로 지시하지 않아도 에이전트가 자동으로 판단합니다.

### 슬래시 커맨드

사용자가 채팅 입력창에 `/skill-name`을 입력합니다. 스킬 본문이 사용자 메시지로 주입되어 에이전트가 해당 스킬의 절차를 즉시 실행합니다.

## 기본 제공 스킬

템플릿에 포함된 스킬들입니다. 프로젝트 생성 시 선택한 템플릿에 따라 복사됩니다.

### 공통 스킬

- **build-renderer** (모든 템플릿): 프로젝트 파일 구조를 분석하여 `renderer.ts`를 자동 생성/수정합니다.

### Memory Chat 스킬

- **journal-search**: BM25 검색으로 `journal.md`에서 과거 사건을 회수합니다.

### RPG Chat 스킬

- **dice-roll**: crypto RNG를 사용한 진짜 주사위 판정을 수행합니다.
- **dynamic-lorebook**: 키워드 기반으로 세계관 데이터를 선택적으로 로드합니다.

### Novel 스킬

- **outline**: 3막 비트 시트로 플롯을 구성합니다.
- **characters**: 캐릭터를 설계합니다.
- **world-building**: 세계관을 구축합니다.
- **chapter-writing**: 아웃라인에 따라 챕터를 집필합니다.
- **revision**: 일관성 검토 및 원고 편집을 수행합니다.
